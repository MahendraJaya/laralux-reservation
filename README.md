## Run Locallcy

Clone git

-   git clone https://github.com/MahendraJaya/laralux-reservation.git
    change the directory
-   cd laralux-reservation
    Install dependencies
-   composer install
-   npm install
-   npm run build
    Set up environment
-   cp .env.example .env
-   php artisan key:generate
-   php artisan migrate
Start server
-   php artisan serve
